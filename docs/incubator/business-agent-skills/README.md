# Business Agent Skill Source Materials

This directory holds non-runtime source materials that were previously placed under `.agents/skills/manga` and `.agents/skills/novel`.

Rules:

- Runtime business-agent Skills are loaded only from `ArtVerse/src/main/resources/business-agent-skills/`.
- `.agents/skills` is reserved for Codex/development skills and must not contain backend runtime business Skills.
- Anything under this directory is reference or incubation material only. Backend runtime code must not scan or depend on it.
