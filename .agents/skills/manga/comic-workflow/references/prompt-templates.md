# 提示词模板

发送给 ChatGPT 的每个提示词都要独立完整：写明 3:4 竖版、画风锚点、主体设计、构图要求、抖音手机竖屏文字大小约束和禁止事项。默认画风见 `visual-style.md`。

## 角色参考图

```text
请生成一张 3:4 竖版漫画角色参考图。

画风：[bible/style.md 的画风锚点]
角色：[姓名]，[故事职能]
固定设计：[体型轮廓、脸部/发型、服装、配饰/道具、色彩重点、表情范围]
构图：全身展示，中性背景，轮廓清晰，便于后续页面复用。
禁止：[不要额外角色、对白气泡、水印；补充角色禁忌]
输出：单张角色参考图。
```

## 场景参考图

```text
请生成一张 3:4 竖版漫画场景参考图。

画风：[bible/style.md 的画风锚点]
场景：[名称]，[叙事作用]
固定设计：[时代/空间布局、关键道具、光线/天气、材质色彩、可复用标志物]
构图：竖版建立镜头，空间关系清楚，前景/中景/背景分层。
禁止：[不要主角、水印；补充场景禁忌]
输出：单张场景参考图。
```

## 漫画页

```text
请根据附件参考图生成这一页漫画，3:4 竖版。

画风：[bible/style.md 的画风锚点]
附件用途：
- [文件名]：[角色/场景] 的唯一设计依据

页面设置：[阅读方向、分镜数量和布局、文字策略]
文字大小：面向抖音手机竖屏观看。以 1080x1440 竖图为基准，任何可读文字不得小于 48px；对白、旁白、说明框正文建议 56px 以上；主标题建议 80px 以上；若实际分辨率不同，最小文字高度不得低于画布高度的 3.3%。所有文字使用粗黑或深色高对比；每格只放短句，单个文字块不超过 18 个汉字，避免小字、密集字、长段落和边缘细字。
故事节拍：[一句话]
逐格分镜：
1. [镜头/景别、角色、动作、背景、情绪、文字]
2. [...]

要求：保持附件角色和场景一致；保留指定分镜布局；文字必须清晰可读并满足最小字号约束，不要压在复杂背景上；不要计划外角色、商标、水印或无关文字；画面内不要出现页码、页脚页码、角标编号、Page XX、p.XX、第 XX 页等页面编号。
输出：单张完整漫画页。
```

## 页面修正

```text
只修正上一张图的这些问题：[列出问题]。
其他保持不变：角色、场景、画风、3:4 比例、分镜布局和抖音手机竖屏大字文字策略。
```
