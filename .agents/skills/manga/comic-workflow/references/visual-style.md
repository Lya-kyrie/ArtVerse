# 默认画风

## 画风锚点

3:4 竖版中文手绘教育漫画信息图，面向抖音手机竖屏观看；暖米色旧纸纹理；粗黑/深棕手绘线稿；水彩和彩铅质感；暖金、赭黄、棕色为主，暗红强调标题和关键词；夸张半卡通人物表情；丰富道具细节；手写中文大标题、手绘分区和大字号说明框。

## 使用要点

- 构图强调知识解释和强对比，可用斜线分区、上下/左右对照、多分镜。
- 人物表情夸张，允许汗滴、红晕、惊讶线、焦虑涂鸦线等漫画符号。
- 文字像手写马克笔/毛笔字；按 1080x1440 竖图基准，任何可读文字不得小于 48px，对白/旁白/说明框正文建议 56px 以上，主标题建议 80px 以上；若生成分辨率不同，最小文字高度不得低于画布高度的 3.3%。标题、对白、旁白和说明框必须粗线、高对比，手机全屏观看也能读清；每格只放短句，单个文字块不超过 18 个汉字，避免密集小字和长段落。
- 说明框与画面融合，边框有粗糙手绘感；文字不要压在复杂纹理或低对比背景上。
- 避免写实照片、3D 渲染、赛博朋克、超级英雄美漫、极简扁平矢量、企业 PPT 插画和冷色霓虹。
